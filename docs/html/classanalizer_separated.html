Detailed Description

A class for analyzing directories in a separate thread.

This class provides a separate thread for analyzing directories at regular intervals. It takes an interval in seconds at which to perform the analysis, a vector of directory paths to analyze, and a pointer to a QTextEdit widget to display the results of the analysis.
